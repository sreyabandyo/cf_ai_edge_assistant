# AI Edge Assistant (Cloudflare Workers)

This is project is an AI-powered backend built using Cloudflare Workers and Workers AI (Llama 3).

## Features
- Clean and responsive web UI for user interaction
- Real-time request/response flow with edge compute
- Serverless deployment on Cloudflare's global network
- Lightweight and fast (no backend servers required)
